# Skills Directory

This folder contains custom skill definitions for the Copilot custom agent repository.

Best practices:
- Use `*.skill.md` files for reusable skills.
- Keep each skill focused on a single capability.
- Include clear metadata and examples if the skill is intended for reuse.
- Use this directory only for skill definitions, not agent prompt content.

If no custom skills are currently required, keep this directory as a placeholder for future extension.
